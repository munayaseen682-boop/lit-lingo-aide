## 📝 Project Report

### a. App Name, Purpose, and Problem Solved
*   **App Name:** LitLingo AI
*   **The Problem:** Students studying English literature and language often struggle to bridge the gap between complex historical literary concepts and practical language application. Traditional study methods lack interactive, instant feedback, making preparation for literature-focused exams or aptitude tests highly stressful and isolating.
*   **The Solution & Target Audience:** LitLingo AI is an independent web application designed specifically for university students and literary enthusiasts. It acts as a 24/7 interactive learning assistant that contextualizes historical English literature movements, analyzes text structures, and provides immediate feedback to elevate user comprehension and language skills.

### c. Comprehensive Features List
*   **Interactive Literature Companion:** Explores major eras, authors, and themes across the history of English literature with adaptive explanations.
*   **Real-time AI Chat Feedback:** Users can submit literary passages or grammar queries to receive immediate analytical breakdowns.
*   **Clean Responsive Interface:** Fully optimized for mobile and desktop screens, allowing students to study seamlessly on the go.

---

### d. The AI Feature & Core Instructions
*   **What it does:** The core application features an embedded AI assistant that processes user prompts, analyzes text complexity, and explains literary devices (such as metaphors, imagery, or historical context).
*   **System Instructions (Behind the AI):**
    > *"You are LitLingo AI, an expert tutor in English literature and linguistics. Your goal is to guide students by breaking down complex literary periods, textual analysis, and grammar into clear, digestible, and academically rigorous explanations. Always maintain an encouraging, educational tone."*

---

### e. Tools, Services, and AI Models Used
*   **Frontend & Application Builder:** Lovable AI platform (React, Vite, Tailwind CSS)
*   **Version Control & Code Hosting:** GitHub
*   **Database & Backend:** Supabase (integrated via Lovable)
*   **Core AI Model:** Anthropic Claude 3.5 Sonnet / OpenAI GPT-4o

### g. How to Run the Project Locally
To run this project on your local machine for evaluation, follow these steps:

1.  **Clone the Repository:**
    ```bash
    git clone [https://github.com/munayaseen682-boop/lit-lingo-aide.git](https://github.com/munayaseen682-boop/lit-lingo-aide.git)
    ```
2.  **Navigate to the Directory:**
    ```bash
    cd lit-lingo-aide
    ```
3.  **Install Dependencies:**
    ```bash
    npm install
    ```
4.  **Start the Local Development Server:**
    ```bash
    npm run dev
    ```
5.  **Open in Browser:** Navigate to `http://localhost:5173` to view the local instance.
